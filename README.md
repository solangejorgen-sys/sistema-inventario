# Sistema de Inventário

Sistema de Inventário moderno usando:

- Vue.js
- PHP API
- PostgreSQL
- Docker

## Executar

```bash
docker compose up --build
```

## URLs

Frontend:
http://localhost

Backend API:
http://localhost:8000

## Funcionalidades

- Adicionar produtos
- Listar produtos
- Editar produtos
- Remover produtos
- Interface moderna
